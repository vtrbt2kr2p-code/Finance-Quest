# Finance Quest 🏆

Application ludique pour apprendre la finance, niveau par niveau — comme un jeu.

## Ce que ça propose
- 12 niveaux progressifs : des bases (épargne, intérêt composé) jusqu'à la finance quantitative
- Leçons guidées avant chaque question, façon professeur particulier
- Glossaire, classeur de formules, fiches de révision (flashcards)
- Système de points, ligues, défi du jour, mode chrono
- Thème clair/sombre, choix de police

## Comment l'utiliser
Ouvre simplement le lien de l'application (GitHub Pages) dans ton navigateur, sur ordinateur ou mobile. Aucune installation requise.

Sur mobile, tu peux l'ajouter à ton écran d'accueil pour qu'elle se comporte comme une vraie appli (voir le menu "Partager" sur iPhone, ou "Installer l'application" sur Android).

## ⚠️ Avertissement
Contenu à but pédagogique uniquement. Ne constitue en aucun cas un conseil en investissement.
